🌑 Dark Web क्या होता है? — सरल भाषा में

Internet के 3 हिस्से होते हैं:

हिस्साक्या हैExample
Surface Web जो Google पर search होता है YouTube, Facebook, News sites
Deep Web जो search में नहीं आता — login required आपकी Gmail, Bank account, Hospital records
Dark Web Special browser (Tor) से accessible — anonymous, untracked Stolen data markets, illegal activities

Dark Web पर — चोरी हुए passwords, credit card numbers, Aadhar data, email lists — यह सब खुलेआम बिकता है। India में हर साल करोड़ों लोगों का data किसी न किसी company से leak होता है — और वो data Dark Web पर पहुँच जाता है। Bigbasket, Mobikwik, Unacademy, AIIMS Delhi — इन सबका data Dark Web पर बिक चुका है।

⚠️
Dark Web पर खुद कभी मत जाओ — यह कारण हैं: Tor browser install करते ही — आपका IP suspicious list में आ जाता है। Dark Web की 70% sites fake हैं — आते ही malware download हो जाता है। Credit score check, data check के नाम पर fake sites — पहले login करवाती हैं, फिर सारा data चुरा लेती हैं। India में Tor browser use करना technically legal है — लेकिन इससे जो sites खुलती हैं उनसे interaction — illegal। एक गलत click — और आपका पूरा device compromise।

☠️ Fake Dark Web Checker — सबसे बड़ा जाल

बहुत से YouTube videos और Telegram posts में लिखा होता है — "यहाँ जाकर check करो — क्या आपका Aadhar Dark Web पर है?" या "यह site free में बताएगी — आपका Credit Score Dark Web पर है?"

यह सब Credit Score और Data Leak का डर दिखाकर traffic लाने की trick है। जैसे ही आप उस site पर जाते हो और email या phone डालते हो — तीन में से एक काम होता है:

पहला — वो आपकी email/phone को Dark Web पर sell कर देते हैं। दूसरा — आपके device पर tracking script चलने लगती है। तीसरा — आपको दिखाते हैं "हाँ आपका data leaked है — ₹499 में remove करवाओ।" यह पूरा fear-based scam है। Data वहाँ था नहीं — आपने खुद दे दिया।

✅ 3 Safe Tools — जो सच में Trusted हैं

🛡️ Tool 1 — HaveIBeenPwned.com (सबसे Trusted)

यह दुनिया का सबसे trusted data breach checker है। इसे Troy Hunt ने बनाया है — Microsoft का Regional Director। FBI और अन्य law enforcement agencies इसे officially use करती हैं। इसमें 14 billion से ज़्यादा leaked accounts का database है।

कैसे use करें:

Browser में टाइप करो: haveibeenpwned.com अपनी email address डालो — कोई password नहीं माँगता यह site। "pwned?" button दबाओ। अगर हरा रंग आया — "Good news" — आपकी email किसी known breach में नहीं मिली। अगर लाल रंग आया — कौन-कौन सी company का breach था — वो दिखेगा। जैसे — "Your email was found in Bigbasket breach (2020) — 20 million records।"

✅
यह Safe क्यों है: यह site सिर्फ email address से check करती है — password नहीं माँगती। Email भी एक special technique (k-anonymity) से check होती है — यानी आपकी पूरी email भी server पर नहीं जाती। Troy Hunt की site को Microsoft, Apple, और governments officially recommend करती हैं। यह non-profit basis पर चलती है — कोई subscription नहीं, कोई trick नहीं।

🛡️ Tool 2 — Google का खुद का Password Checkup

अगर Google Chrome use करते हो या Google account है — तो Google एक free tool देता है जो आपके saved passwords check करता है।

कैसे use करें:

Browser में टाइप करो: passwords.google.com Login करो। "Check Passwords" button दबाओ। Google बताएगा — कौन से passwords किसी data breach में मिले। कौन से passwords weak हैं। कौन से passwords reused हैं। यह सब आपके device पर locally check होता है — Google servers पर नहीं जाता।

🛡️ Tool 3 — Firefox Monitor

Mozilla Firefox का free service। HaveIBeenPwned के same database को use करता है — लेकिन इसमें एक extra feature है।

कैसे use करें:

Browser में टाइप करो: monitor.firefox.com Email डालो — check करो। और एक अच्छी बात — Email Alerts setup कर सकते हो। अगर भविष्य में आपकी email किसी नए breach में मिली — तो Firefox Monitor आपको automatically email कर देगा। यह completely free है।

🔴 Fake Dark Web Checkers — इन्हें पहचानो

यह signs देखो — अगर कोई भी site इनमें से कुछ भी करे — तुरंत बंद करो:

Fake Site का Behaviorअसली मकसदआपको क्या करना है
Password माँगे "check" के लिए 🔴 Password चुराना तुरंत बंद करो
Aadhar / PAN number माँगे 🔴 Identity theft कभी मत दो — कहीं भी
"₹499 में data remove करो" 🔴 Fake fear scam Dark Web से data remove नहीं होता — यह झूठ है

APK file download करवाए "deep scan" 🔴 Malware install Delete करो — रिपोर्ट करो
"Your credit score is on dark web" popup 🔴 Fear से login करवाना Fake है — बंद करो

🆘 Data Leak हो चुका है — अभी यह करो

अगर HaveIBeenPwned ने "Pwned" दिखाया:

पहला काम — Password तुरंत बदलो: जिस site का breach था — उस site का password अभी बदलो। और अगर वही password आपने कहीं और भी use किया है — वहाँ भी बदलो। एक password एक ही जगह — यह rule हमेशा follow करो।

दूसरा काम — 2FA On करो: Two-Factor Authentication — हर important account पर। Gmail, Banking, Social Media। इससे password चोरी होने पर भी account safe रहता है।

तीसरा काम — Bank को Alert करो: अगर financial data leak हुआ है — bank को call करो। "Unusual activity monitor करो।" Debit/Credit card reissue करवाओ अगर card data leaked था।

चौथा काम — Spam calls के लिए तैयार रहो: Phone number leak होने के बाद — scam calls बढ़ जाती हैं। OTP कभी share मत करो — चाहे कोई कितना भी official लगे।

पाँचवाँ काम — Password Manager use करो: Bitwarden (free, open source) — हर account के लिए unique और strong password। एक password याद रखो — बाकी Bitwarden याद रखेगा।

🎯 निष्कर्ष — डर नहीं, जागरूकता
